## Cross-Phase Integration

### Critical Path: Decoder → Live Display

```
AMB Decoder (TCP)
  → Forwarder (Phase 5): AmbRc4TimingSource (Netty) + EpochAnchor
  → :app ForwarderGrpcServer (Phase 5): LapPassingEvent published
  → LapTimingService (Phase 4): processes lap, persists LapTime
  → LiveTimingHub (Phase 4): STOMP broadcast → /topic/race/{id}/timing
  → CockpitPage (Phase 4): live timing table updates
  → useAnnouncements (Phase 6): playBeep(improving)
```

**Status: WIRED** — confirmed across Phase 5 and Phase 6 VERIFICATION.md files.

### Critical Path: Race Lifecycle → Audio Pre-generation

```
RaceStateMachineService.callGrid() (Phase 4)
  → publishes RaceStatusChangedEvent (Phase 6 — retrofitted)
  → AudioPreGenerationService @EventListener (Phase 6)
  → TtsClipService → MinIO storage
  → AudioClipController GET /api/v1/race/{id}/audio-clips
  → usePregeneratedClips (Phase 6): fetches on GRID state
  → playClip() in useAnnouncements (Phase 6)
```

**Status: WIRED** — RaceStatusChangedEvent created in Phase 6 and injected into Phase 4's RaceStateMachineService. Slightly unclean (Phase 6 reaching back into Phase 4) but functionally correct.

### Critical Path: Round Generator → Race Control

```
RoundGeneratorService (Phase 4): creates Round + Race records from entries
  → Race entities consumed by RaceStateMachineService
  → RaceControlController exposes race lifecycle commands
  → LiveRaceState in-memory race representation
  → STOMP broadcasts during race
```

**Status: STRUCTURALLY COMPLETE** — Phase 4 has all 7 plans complete. No VERIFICATION.md to confirm E2E; implementations exist per SUMMARY.md files.

### Critical Path: Practice Timing

```
LapPassingEvent (Phase 5 producer)
  → PracticeTimingService @EventListener (Phase 6)
  → transponder resolution via UserRepository
  → PracticeLap persisted
  → PracticeTimingHub STOMP broadcast → /topic/practice/{id}/timing
  → PracticeLiveTable (Phase 6): live update
```

**Status: WIRED** — Phase 6 VERIFICATION.md confirms PracticeTimingService wired to LapPassingEvent.
