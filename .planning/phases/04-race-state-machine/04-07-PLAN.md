**V19 migration (result_snapshots):**
```sql
CREATE TABLE result_snapshots (
    id            bigserial PRIMARY KEY,
    race_id       bigint NOT NULL UNIQUE REFERENCES races(id) ON DELETE CASCADE,
    finished_at   timestamptz NOT NULL,
    positions_json jsonb NOT NULL,       -- array of ResultRow
    lap_history_json jsonb NOT NULL,     -- array of PositionAtLap

CREATE INDEX idx_result_snapshots_race_id ON result_snapshots(race_id);
```

Store positions and lapHistory as `jsonb` (use `@JdbcTypeCode(SqlTypes.JSON)` on the JPA field, matching V15 `format_overrides` pattern).

**ResultSnapshotService.snapshot(long raceId):**
1. Load Race + Round.
2. Fetch `LiveRaceState` via `lapTimingService.peek(raceId)`; if present, compute `positions` from `state.calculatePositions()` + resolve driverName/carNumber via jOOQ; compute `lapHistory` by walking `state.getLapHistory()` in order and recomputing positions after each passing.
3. If no LiveRaceState (abandoned pre-start), emit an empty positions list and a 0-row lap history.
4. Upsert `result_snapshots` row for this raceId.
5. Release the LiveRaceState from `LapTimingService.states` map (frees memory per Pitfall 2).

**useStomp hook (Pattern 6 from RESEARCH):**
```ts
import { Client } from "@stomp/stompjs";  // already imported by CLAUDE.md stack
export function useStomp(raceId: number | null) {
  const { token } = useAuth();
  const [timing, setTiming] = useState<LiveRacePositionDto[]>([]);
  const [state, setState] = useState<RaceStateChangeDto | null>(null);
  useEffect(() => {
    if (!raceId || !token) return;
    const c = new Client({
      brokerURL: `ws://${window.location.host}/ws/timing`,
      connectHeaders: { Authorization: `Bearer ${token}` },
      reconnectDelay: 5000,
      onConnect: () => {
        c.subscribe(`/topic/race/${raceId}/timing`, (m) => setTiming(JSON.parse(m.body)));
        c.subscribe(`/topic/race/${raceId}/state`, (m) => setState(JSON.parse(m.body)));
      },
    });
    c.activate();
    return () => { c.deactivate(); };
  }, [raceId, token]);
  return { timing, state };
}
```
