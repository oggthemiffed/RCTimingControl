Add private method `applyBumpUpToNextFinal`:
```java
private void applyBumpUpToNextFinal(Race finishedFinalRace, Long eventId) {
    // Only applies if this final has a letter (B, C, ...) — A-Final has no higher final
    String letter = finishedFinalRace.getFinalLetter();
    if (letter == null || letter.charAt(0) <= 'A') {
        // A-Final finishing — nothing to promote; broadcast completion alert instead
        if (liveTimingHub != null) {
            liveTimingHub.broadcastStateChange(finishedFinalRace.getId(), RaceStatus.FINISHED);
        }
        return;
    }

    // Get top N finishers from live state
    Optional<dev.monkeypatch.rctiming.timing.LiveRaceState> state =
            lapTimingService.peek(finishedFinalRace.getId());
    if (state.isEmpty()) {
        log.warn("Bump-up: no live state for finished final race {}", finishedFinalRace.getId());
        return;
    }

    // Determine bumpCount from the next final's RaceEntry bump slots
    // (count entries with bumped=true in the next-higher final)
    String nextLetter = String.valueOf((char)(letter.charAt(0) - 1));
    List<Race> nextFinals = raceRepository.findByEventClassIdAndFinalLetter(
            finishedFinalRace.getEventClassId(), nextLetter);
    if (nextFinals.isEmpty()) {
        log.warn("Bump-up: no {}-final found for event class {}", nextLetter, finishedFinalRace.getEventClassId());
        return;
    }
    Race nextFinal = nextFinals.get(0);
    int bumpCount = (int) raceEntryRepository.findByRaceIdOrderByGridPosition(nextFinal.getId())
            .stream().filter(RaceEntry::isBumped).count();
    if (bumpCount == 0) {
        log.warn("Bump-up: {}-final {} has no bump slots", nextLetter, nextFinal.getId());
        return;
    }

    List<Long> topN = state.get().calculatePositions().stream()
            .limit(bumpCount)
            .map(LiveTimingRowDto::entryId)
            .toList();

    try {
        bumpUpSeedingService.applyBumpUpResults(finishedFinalRace.getId(), topN);
        log.info("Bump-up: promoted top {} from {}-final {} to {}-final {}",
                bumpCount, letter, finishedFinalRace.getId(), nextLetter, nextFinal.getId());
        // Broadcast bump-up alert via STOMP so race director knows before starting next final
        if (liveTimingHub != null) {
            liveTimingHub.broadcastBumpUpAlert(nextFinal.getId(), topN);
        }
    } catch (Exception e) {
        log.warn("Bump-up promotion failed for final race {}: {}", finishedFinalRace.getId(), e.getMessage());
    }
}
```

**Also in `RaceStateMachineService`:**
- Inject `BumpUpSeedingService bumpUpSeedingService` (constructor, @Nullable)
- Inject `RaceEntryRepository raceEntryRepository` (already needed for bump-up count lookup)

const skipTo = useMutation({
