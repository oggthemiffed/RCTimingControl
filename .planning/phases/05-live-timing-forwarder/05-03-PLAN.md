    **E. ForwarderTokenService.java** — `package dev.monkeypatch.rctiming.forwarder;`. `@Service @Transactional`. Constructor-inject `ForwarderTokenRepository repo, PasswordEncoder passwordEncoder`. Use `java.security.SecureRandom` field (single instance).

    Methods:
    ```java
    public record GenerateResult(String plaintext, ForwarderToken persisted) {}

    public GenerateResult generate() {
        repo.findAllByStatus(ACTIVE).forEach(t -> {
            t.setStatus(REVOKED);
            t.setRevokedAt(Instant.now());
        });
        byte[] bytes = new byte[32];
        secureRandom.nextBytes(bytes);
        String plaintext = Base64.getUrlEncoder().withoutPadding().encodeToString(bytes);
        ForwarderToken token = new ForwarderToken();
        token.setTokenHash(passwordEncoder.encode(plaintext));
        token.setStatus(ACTIVE);
        token.setGeneratedAt(Instant.now());
        repo.save(token);
        return new GenerateResult(plaintext, token);
    }

    public void revoke() {
        repo.findFirstByStatusOrderByGeneratedAtDesc(ACTIVE).ifPresent(t -> {
            t.setStatus(REVOKED);
            t.setRevokedAt(Instant.now());
        });
    }

    @Transactional(readOnly = true)
    public Optional<ForwarderToken> validate(String plaintext) {
        if (plaintext == null || plaintext.isBlank()) return Optional.empty();
        return repo.findAllByStatus(ACTIVE).stream()
            .filter(t -> passwordEncoder.matches(plaintext, t.getTokenHash()))
            .findFirst();
    }

    @Transactional(readOnly = true)
    public CurrentStatus getCurrentStatus() {
        var active = repo.findFirstByStatusOrderByGeneratedAtDesc(ACTIVE);
        if (active.isPresent()) {
            return new CurrentStatus(ACTIVE, active.get().getGeneratedAt(), null);
        }
        var any = repo.findFirstByOrderByGeneratedAtDesc();
        return any.map(t -> new CurrentStatus(REVOKED, t.getGeneratedAt(), t.getRevokedAt()))
                  .orElse(new CurrentStatus(null, null, null));  // NONE
    }

    public record CurrentStatus(ForwarderTokenStatus status, Instant generatedAt, Instant revokedAt) {}
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser (admin user) → REST controller | Admin's authenticated session; ADMIN role required |
| REST controller → service / DB | Internal; trusted |
| HTTP response body (one-time-reveal) | Token plaintext crosses this boundary exactly once |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-08 | Spoofing | Forwarder impersonation via stolen token | mitigate | Token is 256-bit `SecureRandom` (43-char Base64URL) — guess-resistant. Stored as BCrypt hash in DB |
| T-05-09 | Tampering | Replay of revoked token | mitigate | `ForwarderTokenService.validate` only iterates `findAllByStatus(ACTIVE)` — revoked tokens are never matched (Plan 04 interceptor calls validate per-stream-connect) |
| T-05-10 | Information Disclosure | Token plaintext logged / returned twice | mitigate | Plaintext returned ONLY in POST response body. GET status DTO contains NO token field — explicit acceptance criterion. Service `generate()` returns plaintext to controller via record; never logged |
| T-05-11 | Information Disclosure | Token plaintext intercepted on venue LAN (no HTTPS) | accept | v1 venue LAN deployment per design decisions; controller Javadoc notes HTTPS required for production |
| T-05-12 | Elevation of Privilege | Non-ADMIN endpoints | mitigate | `@PreAuthorize("hasRole('ADMIN')")` on class; controller test asserts 403 for RACER role |
| T-05-13 | Denial of Service | Repeated token generation churns DB | accept | ADMIN-only endpoint; not internet-facing; no rate limit needed in v1 |
</threat_model>
