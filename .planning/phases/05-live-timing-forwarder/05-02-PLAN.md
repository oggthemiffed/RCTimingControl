<interfaces>
**Phase 4 contract (from app, NOT directly imported):**
```java
public record LapPassingEvent(long raceId, String transponderNumber, long rtcTimeMicros) {}
```
The forwarder MUST NOT import this — it is on the :app classpath only. The forwarder produces a forwarder-local `ParsedPassing` and serializes it as the protobuf `LapPassing` message. Plan 04's gRPC server side translates the proto into `LapPassingEvent` (filling in `raceId` from the currently RUNNING race).

**RC-4 line format (from `docs/AMB_DECODER_PROTOCOL.md`):**
- Each line begins with SOH (0x01), ends with CRLF
- TAB-separated fields
- PASSING: `@\tdecoderId\tseqNum\ttransponderId\ttimeSinceStart\thits\tstrength\tstatus\tcrc`
- STATUS: `#\t...` (heartbeat ~ every 5 seconds)
- Port 5100 (RC-4 firmware < 4.5)
- No FIRST_CONTACT handshake; decoder streams immediately on connect
- No RESEND in text protocol; gaps are logged only

**D-07 epoch formula:**
```
rtcTimeMicros = (decoderEpoch.toEpochMilli() + (timeSinceStart_s - firstTimeSinceStart_s) * 1000) * 1000L
```
where `decoderEpoch = Instant.now()` captured at FIRST PASSING.

**Simulator launch contract:**
- Gradle task `runSimulator` invokes `dev.monkeypatch.rctiming.forwarder.simulator.SimulatorMain`
- CLI flags: `--mode=playback|generative`, `--port=5100` (default), `--speed=1.0` (playback), `--transponders=11111,22222,33333` (generative), `--interval-ms=20000` (generative, per-transponder lap interval)
- Default sample file (playback): `forwarder/src/main/resources/samples/sample-passings.dump`

    public Optional<ParsedPassing> parse(String line) {
        if (line == null || line.isEmpty()) return Optional.empty();
        String[] f = line.split("\t");
        if (f.length < 1) return Optional.empty();
        return switch (f[0]) {
            case "@" -> parsePassing(f);
            case "#" -> Optional.empty();
            default  -> Optional.empty();
        };
    }
