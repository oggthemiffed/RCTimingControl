**RaceFormatConfig.java** — Sealed interface with Jackson annotations (per D-10). CRITICAL: `@JsonTypeInfo` and `@JsonSubTypes` go on the SEALED INTERFACE, NOT on subtypes (RESEARCH.md Pitfall 4):
```java
package dev.monkeypatch.rctiming.domain.format;

import com.fasterxml.jackson.annotation.JsonSubTypes;
import com.fasterxml.jackson.annotation.JsonTypeInfo;

@JsonTypeInfo(use = JsonTypeInfo.Id.NAME, property = "type")
@JsonSubTypes({
    @JsonSubTypes.Type(value = TimedRaceConfig.class,    name = "TIMED"),
    @JsonSubTypes.Type(value = BumpUpConfig.class,       name = "BUMP_UP"),
    @JsonSubTypes.Type(value = PointsFinalsConfig.class, name = "POINTS_FINALS")
})
public sealed interface RaceFormatConfig
    permits TimedRaceConfig, BumpUpConfig, PointsFinalsConfig {}
```

package dev.monkeypatch.rctiming.domain.format;

import io.hypersistence.utils.hibernate.type.json.JsonType;
import org.hibernate.annotations.Type;
import jakarta.persistence.*;
import java.time.Instant;

@Entity
@Table(name = "race_format_templates")
public class RaceFormatTemplate {
    @Id

    private Long id;

    @Column(nullable = false)
    private String name;

    @Type(JsonType.class)
    @Column(columnDefinition = "jsonb", nullable = false)
    private RaceFormatConfig config;

    @Column(name = "created_at", nullable = false, updatable = false)
    private Instant createdAt = Instant.now();

    @Column(name = "updated_at", nullable = false)
    private Instant updatedAt = Instant.now();

}

@Entity
@Table(name = "event_classes")
public class EventClass {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Type(JsonType.class)
    @Column(name = "config_snapshot", columnDefinition = "jsonb", nullable = false)
    private RaceFormatConfig configSnapshot;

    @Type(JsonType.class)

    @Column(name = "updated_at", nullable = false)
    private Instant updatedAt = Instant.now();
    // getters/setters
}

**RaceFormatService.java** — `@Service @Transactional`:
- Constructor injection: `RaceFormatTemplateRepository`, `EventClassRepository`, `ObjectMapper` (the default Spring-provided one, NOT the yamlObjectMapper)
- `RaceFormatConfig getEffectiveConfig(EventClass eventClass)` — merges snapshot + override:
  1. If `configOverride` is null or empty, return `configSnapshot` directly
  2. Otherwise: serialize `configSnapshot` to a `Map<String, Object>` via ObjectMapper
  3. Overlay all entries from `configOverride` onto the map (override values win)
  4. Deserialize the merged map back to `RaceFormatConfig` via ObjectMapper
  5. Return the merged config
- `EventClass assignTemplateToEventClass(RaceFormatTemplate template)` — creates EventClass with `configSnapshot` = deep copy of `template.getConfig()` (serialize then deserialize to break reference), `template` reference stored for audit (per FORMAT-06)

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| JSON/YAML input → sealed interface | Untrusted config data deserialized into typed Java objects |
| Override map → config merge | Arbitrary key-value pairs merged onto typed config |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-06 | Tampering | RaceFormatConfig deserialization | mitigate | Sealed interface with `@JsonSubTypes` rejects unknown type discriminators; `@Valid` on controller (Plan 05) validates field constraints |
| T-01-07 | Tampering | configOverride merge | mitigate | `@JsonIgnoreProperties(ignoreUnknown = true)` drops unknown fields during merge; override keys that don't exist in the target type are silently ignored |
| T-01-08 | Denial of Service | Large JSONB payloads | accept | PostgreSQL JSONB has practical size limits; no additional payload size enforcement for admin-only endpoints (low risk for single-club deployment) |
</threat_model>
