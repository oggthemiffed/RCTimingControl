All imports MUST use `jakarta.persistence.*` (NOT javax — Hibernate 6 / Spring Boot 3.x). All Spring beans use constructor injection (no @Autowired field injection).

**Role.java** — Java enum (per D-04 in domain/user/ package, per AUTH-05):
```java
package dev.monkeypatch.rctiming.domain.user;
public enum Role { RACER, ADMIN, RACE_DIRECTOR, REFEREE }
```

**User.java** — JPA entity mapped to `users` table. Per AUTH-05, roles are stackable via ElementCollection:
- `@Entity @Table(name = "users")`
- Fields: `id` (Long, BIGSERIAL via GenerationType.IDENTITY), `email` (unique, not null), `passwordHash` (not null), `firstName`, `lastName`, `createdAt`, `updatedAt` (both TIMESTAMPTZ mapped to `Instant`)
- Roles: `@ElementCollection(fetch = FetchType.EAGER) @CollectionTable(name = "user_roles", joinColumns = @JoinColumn(name = "user_id")) @Enumerated(EnumType.STRING) @Column(name = "role") private Set<Role> roles = new HashSet<>()`
- Standard getters/setters. No Lombok in Phase 1 (keep explicit).

**UserRepository.java** — extends `JpaRepository<User, Long>`:
- `Optional<User> findByEmail(String email)`
- `boolean existsByEmail(String email)`

**UserService.java** — `@Service @Transactional`:
- Constructor injection of `UserRepository` and `PasswordEncoder`
- `createRacer(String email, String password, String firstName, String lastName)` — hash password with PasswordEncoder, assign `Role.RACER`, save, return User
- `findByEmail(String email)` — returns Optional<User>
- Check duplicate email via `existsByEmail` before save; throw `DataIntegrityViolationException` or let the unique constraint handle it

**RefreshToken.java** — `@Entity @Table(name = "refresh_tokens")`:
- Fields: `id`, `user` (@ManyToOne to User), `tokenHash` (VARCHAR 64, unique), `expiresAt` (Instant), `createdAt` (Instant), `revoked` (boolean, default false)

**RefreshTokenRepository.java** — extends `JpaRepository<RefreshToken, Long>`:
- `Optional<RefreshToken> findByTokenHash(String tokenHash)`
- `void deleteByUser(User user)` — for revocation on password reset
- `List<RefreshToken> findByUserAndRevokedFalse(User user)`

**PasswordResetToken.java** — `@Entity @Table(name = "password_reset_tokens")`:
- Fields: `id`, `user` (@ManyToOne to User), `tokenHash` (VARCHAR 64, unique), `expiresAt` (Instant), `createdAt` (Instant), `used` (boolean, default false)

**PasswordResetTokenRepository.java** — extends `JpaRepository<PasswordResetToken, Long>`:
- `Optional<PasswordResetToken> findByTokenHash(String tokenHash)`
